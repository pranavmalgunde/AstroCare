
# AstroCare (NeuraVia Hacks Demo)

Real-time, browser-based health companion — Detect · Connect · Personalize.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Tech
React 18, Vite, Tailwind (CDN), Recharts, lucide-react, Web Audio API, getUserMedia, inline SVG.
